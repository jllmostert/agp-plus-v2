🎯 Wat is Phase 5?

Phase 5A: Automatic Locks

Sensors ouder dan 30 dagen krijgen automatisch een lock 🔒. Dit beschermt historische data tegen accidenteel verwijderen.

Phase 5B: Manual Lock Toggles

Lock icons zijn klikbaar - één klik toggle tussen 🔒 (locked) en 🔓 (unlocked). DELETE button respecteert lock status.

Phase 5C: Smart Sensor Sync

Bij app startup worden recente sensors (≤30 dagen) automatisch naar localStorage gesynct. Oude sensors blijven in SQLite database (read-only).
